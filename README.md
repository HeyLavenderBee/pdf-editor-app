# PDF Editor App
Um aplicativo feito com React Native + Expo para facilmente criar e editar arquivos PDF.
Para armazenar os dados do usuário, é usado o SQLite, que permite guardar informações localmente em tabelas.
<br>
Você pode criar arquivos e adicionar imagens da galeria, ou imagens tiradas da câmera dentro do aplicativo.
